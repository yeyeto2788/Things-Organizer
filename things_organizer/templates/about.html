What is this?

This is a common question I get from my colleagues and here it goes a quick (I hope) explanation of what this application is and what is it for.

The idea came from a day I was looking a SOC board like the Raspberry Pi and I could not find it due to the mess I had, all boards, cables, equipment, etc. where all around in different boxes and different places in my house from my room lab to the garage.

So I came up with the idea of develop this application using Flask and other Python libraries that would let me store information about all the things I have and where they are in my house.

Doing it I have faced some challenges since I had no clear idea of what to add on the application or not so I tried to make it so everybody could use it.

Now, I can finally find the thing I'm looking for on the application and it will tell me where I have stored that thing. I can also print the label for that item or even for the storage boxes and bags I have on different locations(e.g. Garage, Room, etc.).

Is it Open Source?

Yes!, from the beggining I wanted everybody to be able to use the application and even modify it to his/her needs.

The code is hosted on a GitHub repository and you can fork the repository and add more features to it or you can even create an Issue and I'll be more that happy to add more functionality to this application.
